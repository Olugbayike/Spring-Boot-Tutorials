# Spring Boot Tutorials
Spring boot coursework from the Hyperskill (Jetbrains Academy) Java backend course

# Installation
Install JDK 25 from [Oracle](https://www.oracle.com/java/technologies/downloads/#java25)

# Usage
## customUserStore 
run the main method in CustomUserStoreApplication.java file

## jpaProjections
run the main method in JpaProjectionsApplication.java file

## jpaRepository
run the main method in JpaRepositoryApplication.java file

## springSecurity
run the main method in SpringSecurityApplication.java file

## springSecurityTest
run the main method in SpringSecurityTestApplication.java file

## springTest
run the main method in SpringTestApplication.java file

## springWebMvcTutorials
run the main method in SpringWebMvcTutorialsApplication.java file

## userDetailsServiceImplementation
run the main method in UserDetailsServiceImplementationApplication.java file
